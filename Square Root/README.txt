Square Root v. 1.0

Rebecca Magier and Julian Segert for Bio 119: Genomics and Systems Biology S16

To use, add this folder to Fiji.app/plugins/Scripts/Plugins, drag bright.model
 to the outer Fiji.app directory and restart Fiji.

Requires the Image Padder plugin by Frederic V. Hessman of
Institut für Astrophysik, Georg-August-Universität Göttingen:

http://www.astro.physik.uni-goettingen.de/~hessman/ImageJ/Image_Padder/
